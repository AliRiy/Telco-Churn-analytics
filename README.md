# Telco Customer Churn analytics
This project is a data analytics and predictive modeling workflow built using Python, SQL, Pandas, NumPy, Matplotlib, Seaborn, Excel, and Tableau. \
\
The goal is to **analyze customer behavior, identify churn drivers, build predictive models, and generate actionable business insights** for a subscription-based telecom business.

### The project covers:
-	Data ingestion & SQL database design
-	Data cleaning and preprocessing
-	Exploratory data analysis (EDA) and visualization
-	Feature engineering
-	Customer segmentation (K-means + revenue profiling)
-	Churn prediction modeling (Logistic Regression, Random Forest)
-	Revenue-at-risk analysis and actionable recommendations
-	Tableau dashboard + Excel stakeholder deliverables

### Tools & Technologies:
- Python: Pandas, NumPy, Matplotlib, Seaborn, Scikit-Learn
- SQL: SQLite/PostgreSQL
- Visualization: Tableau or Power BI
- Excel: Pivot tables, charts, KPI summaries
- Jupyter Notebooks: Full workflow documentation

### Business Objective:
- Understand why customers churn
- Identify which customers are most likely to leave
- Recommend how to reduce churn to protect recurring revenue
